# Calculadora PythonEste repositorio contiene una calculadora básica desarrollada en Python, con funciones para realizar operaciones aritméticas simples.## Estructura del proyecto- **main.py**: Archivo principal que ejecuta el menú de opciones.- **sumar.py**: Contiene la función para sumar dos números.- **resta.py**: Contiene la función para restar dos números.- **multiplicacion.py**: Contiene la función para multiplicar dos números.- **dividir.py**: Contiene la función para dividir dos números.- **suma_avanzada.py**: Contiene la función para sumar una cantidad N de números.## Requisitos previos- Python 3.6 o superior instalado.## Cómo ejecutar1.
